## üöÄ Getting Started

### Prerequisites
- Java 11 or higher
- Maven 3.6+ or Gradle 6+
- IDE (IntelliJ IDEA, Eclipse, or VS Code)

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Complete-Java-Masterclass.git
   ```

2. Navigate to specific project directories and run:
   ```bash
   # For Maven projects
   mvn clean compile
   
   # For Gradle projects
   ./gradlew build
   ```
